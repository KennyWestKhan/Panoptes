# Panoptes
CMS with OCR

A Content Management System with Optical Character Recognition feature to read text from a picture
# Server setup
```
npm install
```
# Client setup
```
cd client/
```
```
npm i
```
> go back to root folder of project
# Project run
```
npm run dev
```
Test Demo [here](https://morning-journey-40183.herokuapp.com/).
